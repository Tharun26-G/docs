# 1. Conditional Statements (`if`, `elif`, `else`)

Used to execute different code based on conditions.

## Syntax

```python
if condition:
    # code
elif condition:
    # code
else:
    # code
```

---

## `if`

Runs only if the condition is `True`.

```python
age = 20

if age >= 18:
    print("Adult")
```

Output

```text
Adult
```

---

## `if...else`

```python
age = 15

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

Output

```text
Minor
```

---

## `if...elif...else`

Checks multiple conditions.

```python
marks = 85

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")
```

Output

```text
Grade B
```

---

## Nested if

```python
age = 20
citizen = True

if age >= 18:
    if citizen:
        print("Eligible to vote")
```

Output

```text
Eligible to vote
```

---

## Short Hand if

```python
a = 10

if a > 5: print("Greater")
```

---

## Short Hand if...else (Ternary Operator)

```python
a = 10

print("Even") if a % 2 == 0 else print("Odd")
```

---

## Logical Operators

### `and`

Both conditions must be True.

```python
age = 25
salary = 40000

if age > 18 and salary > 30000:
    print("Eligible")
```

---

### `or`

At least one condition must be True.

```python
age = 15
student = True

if age > 18 or student:
    print("Allowed")
```

---

### `not`

Reverses the result.

```python
logged_in = False

if not logged_in:
    print("Please login")
```

---

## Comparison Operators

| Operator | Meaning |
| :-- | :-- |
| `==` | Equal |
| `!=` | Not equal |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater or equal |
| `<=` | Less or equal |

---

## Membership Operators

```python
fruits = ["Apple", "Banana"]

if "Apple" in fruits:
    print("Found")
```

---

## Identity Operators

```python
a = [1,2]
b = a

print(a is b)
```

Output

```text
True
```

---

# Quick Revision

### Conditional Statements

- `if` → Check one condition.
- `elif` → Check another condition.
- `else` → Runs if all conditions are False.
- Nested `if` → `if` inside another `if`.
- Ternary operator → Short `if-else`.
