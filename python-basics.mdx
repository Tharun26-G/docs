# 1. Variables

### Definition

A variable is used to store a value.

### Example

```python

```

### Output

```text
(No output)
```

### Explanation

Variables store data that can be used later in the program.

---

# 2. Variable Naming Rules (snake\_case)

### Rules

- Use letters, numbers, and underscores.
- Cannot start with a number.
- Case-sensitive.
- Use `snake_case` for readability.

### Good Examples

```python
student_name = "Alice"
total_marks = 95
```

### Bad Examples

```python
2name = "Alice"
student-name = "Alice"
```

### Output

```text
SyntaxError
```

### Explanation

Use meaningful names written in lowercase with underscores.
